Dive Into Python

Python per programmatori esperti

Dive Into Python è un libro gratuito sul linguaggio di programmazione Python, per programmatori esperti. Potete leggere il libro online o scaricarlo in uno dei vari formati disponibili. È inoltre disponibile in svariate lingue.

Leggi Dive Into Python

Questo libro è ancora in fase di scrittura. I primi tre capitoli sono una solida infarinatura sulla programmazione in Python. I capitoli riguardanti l'elaborazione HTML, l'elaborazione XML, e i test delle unità di codice sono completi, un capitolo riguardante la programmazione funzionale è in sviluppo. Questo non è un sito fantoccio per grandi lavori commerciali; tutti i nuovi contenuti saranno pubblicati qui, gratis, appena pronti. Potete leggere la cronologia delle revisioni per vedere cosa c'è di nuovo. Updated 30 gennaio 2003
