1.15. Заключение

Теперь программа odbchelper.py и ее вывод приобретают смысл.

Пример 1.37. odbchelper.py


def buildConnectionString(params):
    """Создает и возвращает строку соединения из словаря параметров."""
    return ";".join(["%s=%s" % (k, v) for k, v in params.items()])

if __name__ == "__main__":
    myParams = {"server":"mpilgrim", \
                "database":"master", \
                "uid":"sa", \
                "pwd":"secret"}
    print buildConnectionString(myParams)

Пример 1.38. Вывод программы odbchelper.py

server=mpilgrim;uid=sa;database=master;pwd=secret

Перед тем, как погрузиться в чтение второй главы, убедитесь, что вы овладели следующими навыками:

  • Использование Python IDE в интерактивном режиме для проверки выражений
  • Написание модуля Python таким образом, чтобы он мог быть запущен как отдельная программа, хотябы с целью тестирования
  • Импортирование модуля и вызов его функций
  • Определение функций, использование строк документации и локальных переменных, использование отступов для выделения блоков кода
  • Определение словарей, кортежей и списков
  • Доступ к атрибутам и методам произвольных объектов, включая строки, списки, словари, функции и модули
  • Получение форматированного представления
  • Преобразование списков с помощью расширенной записи списков
  • Разбиение строк на подстроки и их объединение
